# 使用者活動報告顯示名稱突然變成大寫英文+數字，是否有辦法更改回中英文姓名顯示？

* 前往Microsoft 365 admin center https://admin.microsoft.com/ <br>
  (一) 至左邊選單中點選  <br>
   **1.設定**  <br>
   **2.組織設定** <br>
   **3.報告** <br>
